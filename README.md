## Installation

```bash
$ yarn install

```

# Notes
The hardcoded base URL from the provided API needs to be changed manually at ./src/services/api.
We do recommend using NGROK for tunneling localhost.
IPFS takes some time to load images, be patient.
This is an EXPO managed application, so it's deeplink goes for EXPO, you need to install it first on your mobile device for the application to run properly.

## Running the app

```bash
# development
$ yarn run start

```
